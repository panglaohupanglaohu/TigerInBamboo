# 参考文献与画作

## 论文

- **Tu, Xiaoyuan & Terzopoulos, Demetri. _Artificial Fishes: Physics, Locomotion, Perception, Behavior_.**
  本平台的智能体框架蓝本：生物作为自主智能体，具备感知（perception）、
  内驱力（hunger / fear / libido）、行为库（behavior repertoire）与运动控制（motor control）。
  配置页中的"物种关系矩阵"即对应论文中的 predator–prey、逃避（evasive action）、
  集群（schooling）等关系模型。原文 PDF 见用户提供的 `192161.192170.pdf`（ACM Digital Library 编号）。
